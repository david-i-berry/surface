= Images Directory

This directory contains screenshots and diagrams for the SURFACE documentation.

== Image Guidelines

=== File Naming Convention

Use descriptive, lowercase names with hyphens:

----
<section>-<feature>-<description>.png
----

Examples:

* `user-map-layer-controls.png`
* `admin-station-create-form.png`
* `dev-architecture-overview.png`

=== Recommended Formats

[cols="1,2,2"]
|===
|Format |Use Case |Notes

|PNG
|Screenshots, UI elements
|Preferred for most images

|SVG
|Diagrams, flowcharts
|Scalable, editable

|JPG
|Photos
|Use for station photos only
|===

=== Image Dimensions

* *Screenshots:* Capture at 1920x1080 or higher, will be scaled down
* *Diagrams:* Export at 2x resolution for retina displays
* *Maximum file size:* 500KB per image (optimize with tools like `pngquant`)

== Extracting Images from Existing Manual

The existing User Manual (`docs/SURFACE User Manual - working draft.md`) contains embedded images.
To extract them:

1. The images are base64 encoded in the markdown
2. Use a script to extract and save as PNG files
3. Rename according to the naming convention above
