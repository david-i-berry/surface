= API Reference
:imagesdir: ../images

This chapter documents the SURFACE REST API endpoints.

== API Overview

SURFACE provides a RESTful API for programmatic data access. All responses are in JSON format.

=== Base URL

----
https://<your-surface-instance>/api/
----

Example: `https://dev.nms.gov.bz/api/`

== Authentication

SURFACE supports multiple authentication methods for API access.

=== JWT Authentication (Recommended)

JSON Web Tokens provide secure, stateless authentication.

==== Obtain Token

[source,http]
----
POST /api/token/
Content-Type: application/json

{
    "username": "your-username",
    "password": "your-password"
}
----

Response:
[source,json]
----
{
    "access": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
    "refresh": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..."
}
----

==== Using JWT Token

Include the access token in all API requests:
[source,http]
----
Authorization: Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...
----

==== Refresh Token

Access tokens expire after 24 hours. Use the refresh token to obtain a new access token:

[source,http]
----
POST /api/token/refresh/
Content-Type: application/json

{
    "refresh": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..."
}
----

=== DRF Token Authentication

Django REST Framework token authentication (legacy method).

[source,http]
----
Authorization: Token abc123xyz...
----

Tokens are managed in Django Admin > *Auth Token* > *Tokens*.

=== Session Authentication

For browser-based access, standard Django session cookies are used after login through the web interface.

== Station Endpoints

=== List Stations

[source,http]
----
GET /api/stations/
----

Returns list of all stations with metadata.

==== Response Fields

[cols="1,3"]
|===
|Field |Description

|id
|Database ID

|code
|Station code

|name
|Station name

|latitude
|Decimal degrees

|longitude
|Decimal degrees

|elevation
|Meters above sea level

|is_active
|Boolean status

|is_automatic
|Boolean (AWS vs manual)
|===

=== Get Station Details

[source,http]
----
GET /api/stations/{id}/
----

Returns detailed metadata for specific station.

== Variable Endpoints

=== List Variables

[source,http]
----
GET /api/variables/
----

Returns all measurement variables defined in the system.

==== Response Fields

[cols="1,3"]
|===
|Field |Description

|id
|Variable ID (use in data queries)

|name
|Variable name

|symbol
|Short symbol

|unit
|Unit of measurement
|===

== Live Data

=== Get Live Data

[source,http]
----
GET /api/livedata/{station_id}/
----

Returns current/recent data for a station. Updated every 30 minutes.

==== Response Sections

[cols="1,3"]
|===
|Section |Description

|latest
|Most recent raw observation

|latestdaily
|Current day's summary (00-24h local)

|past24h
|Rolling 24-hour summary
|===

== Raw Data

=== Query Parameters

[cols="1,3"]
|===
|Parameter |Description

|search_type
|`station`, `variable`, or `stationvariable`

|search_value
|ID for search type

|search_value2
|Second ID (for stationvariable)

|search_date_start
|Start datetime (ISO 8601)

|search_date_end
|End datetime (ISO 8601)
|===

=== Search by Station

[source,http]
----
GET /api/rawdata/?search_type=station&search_value={station_id}
----

Maximum: 7 days of data per request.

Default (no dates): Last 7 days.

==== With Date Range

[source,http]
----
GET /api/rawdata/?search_type=station&search_value=122&search_date_start=2024-05-10T00:00:00Z&search_date_end=2024-05-13T00:00:00Z
----

=== Search by Variable

[source,http]
----
GET /api/rawdata/?search_type=variable&search_value={variable_id}
----

Returns specified variable across all stations.

=== Search by Station and Variable

[source,http]
----
GET /api/rawdata/?search_type=stationvariable&search_value={station_id}&search_value2={variable_id}
----

== Summary Data

Summary endpoints aggregate raw data into time periods.

=== Hourly Summaries

[source,http]
----
GET /api/hourlysummaries/?search_type=station&search_value={station_id}
----

Maximum: 30 days per request. Default: Last 32 days.

==== With Date Range

[source,http]
----
GET /api/hourlysummaries/?search_type=station&search_value=122&search_date_start=2024-05-01T01:00:00Z&search_date_end=2024-05-14T00:00:00Z
----

=== Daily Summaries

[source,http]
----
GET /api/dailysummaries/?search_type=station&search_value={station_id}
----

Maximum: 1 year per request. Default: Last 365 days.

=== Monthly Summaries

[source,http]
----
GET /api/monthlysummaries/?search_type=station&search_value={station_id}
----

=== Yearly Summaries

[source,http]
----
GET /api/yearlysummaries/?search_type=station&search_value={station_id}
----

=== Last 24 Hours Summary

[source,http]
----
GET /api/last24hrsummaries/?search_type=station&search_value={station_id}
----

Returns summary statistics for the last 24 hours (rolling window).

[source,http]
----

[source,http]
----

[source,http]
----

GET /api/station_variable_data_month_inventory/?station_id={id}&variable_id={id}

[source,http]

== Rate Limiting

Recommended limits:

* Maximum 60 requests per minute
* Use date ranges to reduce request count
* Cache responses when possible

== Code Examples

=== Python

[source,python]
----
import requests

base_url = "https://dev.nms.gov.bz/api"
token = "your-token-here"

headers = {
    "Authorization": f"Token {token}"
}

# Get stations
response = requests.get(f"{base_url}/stations/", headers=headers)
stations = response.json()

# Get raw data for station
params = {
    "search_type": "station",
    "search_value": 122,
    "search_date_start": "2024-05-01T00:00:00Z",
    "search_date_end": "2024-05-07T00:00:00Z"
}
response = requests.get(f"{base_url}/rawdata/", headers=headers, params=params)
data = response.json()
----

=== JavaScript

[source,javascript]
----
const baseUrl = "https://dev.nms.gov.bz/api";
const token = "your-token-here";

const headers = {
    "Authorization": `Token ${token}`
};

// Get live data
fetch(`${baseUrl}/livedata/122/`, { headers })
    .then(response => response.json())
    .then(data => console.log(data));
----

=== cURL

[source,bash]
----
# Get stations
curl -H "Authorization: Token your-token" \
    https://dev.nms.gov.bz/api/stations/

# Get daily summaries
curl -H "Authorization: Token your-token" \
    "https://dev.nms.gov.bz/api/dailysummaries/?search_type=station&search_value=122"
----
